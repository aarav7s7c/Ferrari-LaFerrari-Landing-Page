# LaFerrari Landing Page

A tribute landing page for the Ferrari LaFerrari — designed in Figma and hand-coded from scratch.

## How it's made
**Tech used**: HTML, CSS and JS.
**Colors Primarily used**: *Racing Red* `#EF1A2D`, *Bone* `#DFDCCD`, *Ink Black* `#020E12`, 
*Grey Olive* `#838782`.

No template, no framework used, no AI code generation used. **Complete handwritten code**. For the designing I used *Figma* and coded the Landing Page on *VS code*.
Every spacing, every width, every size of the font, is *selected with a purpose*. All the sizes are mainly used with the rule of **8px**.

Link to project: https://aarav7s7c.github.io/Ferrari-LaFerrari-Landing-Page/

## Learning
Making this page, i faced various issues and overcame them. Many parts of the website took hours to make but not giving up was my best resolve. I may not have used Ai to generate code, but I believe it was totally worth it writing the code with my own keyboard, many pages of my notebook are filled with the design concept and code ideas. Yet there is no part in the site which i left as "Meh, it's enough", every minor problem i encountered, i solved even when it got me frustrated.
